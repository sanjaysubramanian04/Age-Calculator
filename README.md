# Age Calculator 🕒

A simple, responsive web application that calculates your age based on your birthdate. Built with **HTML, CSS, and JavaScript**, this project provides an interactive, mobile-friendly interface for quick age calculation.

---

## 🌟 Features

- Calculates exact age (years, months, days) from the date of birth.
- Fully **responsive UI**, works perfectly on mobile and desktop.
- Validates user input for proper birthdates.
- Smooth animations and clean design for an engaging experience.

---

## 🛠️ Built With

- HTML5  
- CSS3 (with responsive design)  
- JavaScript (vanilla)
